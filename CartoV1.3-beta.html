// Mode opératoire

1. Position récepteur : cliquez sur la carte ou saisissez les coordonnées.

2. Azimut : entrez le relèvement magnétique/géographique en degrés (0–360°).

3. Incertitude : le secteur angulaire d'incertitude sera représenté en transparence.

4. Longueur du trait : longueur du tracé sur la carte.

5. QRK : force du signal, modifie l'épaisseur du trait de relevé.

// Clé API IGN (Géoplateforme)

Nécessaire pour les couches privées IGN (Scan 25…).
Créez votre compte gratuit sur geoservices.ign.fr.
Clé temporaire ign_scan_ws utilisée par défaut.
